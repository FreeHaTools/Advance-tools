# Advance Tools

Plugin-based smart home toolbox for Home Assistant with a modern admin hub.
Every feature is a **tool plugin** — Dashboard Maker and Automation Maker ship
built in, and more tools are on the way.

**Dashboard Maker** — build wall-tablet dashboards with plain HTML or the
visual designer and give every family member their own login.
**Automation Maker** — browse, run, enable/disable and visually build HA
automations (trigger / condition / action).
**Entity Doctor** — scan every entity for unavailable, low-battery, orphaned,
stale and duplicate-named problems, then fix them with one click.
**Scene Maker** — snapshot a room exactly as it is now (lights, climate,
covers…), fine-tune it, test it live without saving, then save it as a scene.
**Backup Manager** — see, create, schedule and download Supervisor backups,
with automatic cleanup of old scheduled ones.

- 🖌️ Design dashboards in HTML/CSS/JS — built-in editor or file upload
- 👥 Users & passwords with per-dashboard access control
- 🔒 Per-dashboard entity allowlist (a room's tablet only controls that room)
- 📟 Kiosk-friendly: fullscreen, no back-navigation, 1-year sessions
- ⚡ Live entity states + control over WebSocket (no HA token in the browser)

Open `http://YOUR_HA_IP:8234/`, pick a dashboard, log in. The first run walks
you through creating your own admin password — there is no default one. Full
guide in the Documentation tab.

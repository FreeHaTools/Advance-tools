Installed packs

Packs add card styles and control skins to the Designer. Built-in packs are always available; imported ones are listed below.

Import a pack

Paste the pack JSON, or choose a .json file. The format is documented in Build your own.

The Advance Tools Pack format

A pack is one JSON file holding any mix of card styles and skins (switches, sockets, sensors, …). Import it in the Packs tab.
{
  "pack": "advance-tools-pack",   ← required, exactly this
  "format": 1,
  "name": "My First Pack",
  "author": "you",  "version": "1.0.0",
  "items": [ …styles and skins… ]
}

Example 1 — a card style

.CARD is the card placeholder; .CARD.on = active state. Any CSS works, including @keyframes (unique names).
{ "kind": "style", "id": "lava", "name": "Lava", "category": "My styles",
  "css": ".CARD { background:linear-gradient(135deg,#ff2d00,#ff9500); border:0; }
          .CARD.on { box-shadow:0 0 28px #ff9500; }" }

Example 2 — a skin (a round smart button)

.SKIN is your control's wrapper. "for" decides behavior: toggle light sensor climate button vacuum cover valve media clock. Tokens {{name}} {{icon}} {{val}} get filled; hook classes (at-name at-val at-ico…) get live updates; state classes (on, open, cleaning…) are toggled automatically. "card": true = sits on a card that follows the Card Styles gallery.
{ "kind": "skin", "id": "my-round", "name": "My round button",
  "category": "My switches", "for": "toggle", "size": [150, 160],
  "html": "<div class='b'>{{icon}}</div><div class='at-name'>{{name}}</div>",
  "css": ".SKIN { display:flex; flex-direction:column; align-items:center;
                  justify-content:center; gap:8px; }
          .SKIN .b { width:70%; aspect-ratio:1; border-radius:50%; background:#222;
                     display:flex; align-items:center; justify-content:center;
                     font-size:30px; transition:.25s; }
          .SKIN.on .b { background:var(--accent); box-shadow:0 0 20px var(--accent); }
          .SKIN .at-name { font-size:12px; color:#8b98b8; }" }

Live hooks reference

Text hooks: .at-name .at-val .at-ico .at-bat .at-title
  .at-cur .at-tgt .at-tgt2 .at-nm   .at-arc = SVG arc (dasharray 0-100)
Action hooks: climate .at-minus/.at-plus · vacuum .at-start/.at-pause/.at-dock
  cover .at-up/.at-stop/.at-down · media .at-play/.at-prev/.at-next/.at-vup/.at-vdn
  light .at-bri (range 0-100), tap target .at-main
State classes: on unavail open closed opening closing cleaning docked
  returning playing heat cool low
CSS vars: var(--accent) · var(--pct) 0-100 (fill level)
Tip: every built-in uses this exact format — open /static/packs.js on this server and copy one as a starting point.

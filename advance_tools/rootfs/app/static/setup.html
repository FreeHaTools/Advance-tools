👋 Welcome to Advance Tools

Advance Tools turns your Home Assistant into a set of simple, family-friendly panels — dashboards, alarms, schedules and reminders that anyone in the house can use from a phone or a wall tablet. This short setup takes about two minutes and gets you a secure, working panel straight away.

  • 🔐Create your password Replace the temporary one so nobody else can control your home.
  • 🎨Pick a starter layout We build a first dashboard from your existing devices. Optional.
  • 🚀Start using it Open your panel, explore the tools, read the built-in manual.

🔐 Create your password

Advance Tools can unlock doors, disarm your alarm and control everything in your home. Right now it is protected by a temporary password that everyone knows — please choose a real one before you go any further.

Password strength—
  • ○At least 8 characters
  • ○Not a commonly used password
  • ○Different from your username
Both fields must match.

🎨 Pick a starter layout

Choose a ready-made dashboard and we will fill it with the devices you already have. You can change everything later — or skip this and start from a blank panel.

Starter layouts are not available on this installation yet — no problem, you can build a dashboard any time from the Dashboard Maker tool.

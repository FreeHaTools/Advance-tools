How announcements work

What is TTS?

Text-to-speech (TTS) turns the text you type into spoken audio and plays it on your media players — Google/Nest speakers, Sonos, Alexa (via integrations), TVs, and more. That's what makes this page a whole-house intercom.

Adding a TTS engine (2 minutes)

  • In Home Assistant open Settings → Devices & Services → Add integration.
  • Search for Google Translate text-to-speech — it's free and needs no account.
  • Other options: Home Assistant Cloud (Nabu Casa, very natural voices), Piper (local, private), ElevenLabs.
  • Come back here and hit refresh — the engine appears in the dropdown automatically.

Speakers that are off or unavailable

A speaker must be reachable to play audio. Tiles show a live state badge — unavailable speakers will fail (you'll get a clear error toast for exactly that speaker, the rest still play). Many smart speakers wake up from idle/off automatically when TTS arrives.

Intercom ideas

  • Announce on all speakers when dinner is ready — one tap with a quick message.
  • Bedtime countdown: "10 minutes to lights out" to the kids' rooms.
  • "Car is leaving in 5 minutes" to the whole house before school.
  • Pair with Also send to phones so family members away from home get it as a push notification too.

Dashboard Intercom widget

Dashboards made with Dashboard Maker can talk too: any logged-in dashboard user gets intercom endpoints, so a wall tablet becomes a one-tap intercom station. No admin login needed on the tablet.

  • Quick messages — the same chips you save here appear on the tablet for one-tap TTS announcements.
  • Rooms — speakers are grouped by Home Assistant areas, so the tablet shows "Kitchen", "Bedroom"… instead of entity ids.
  • Voice clips — hold to record, release to play the real recording on the chosen speakers. Browsers only allow microphone access over HTTPS, so tablets should open the dashboard via your https domain. Plain-HTTP tablets still get text and quick-message TTS.
  • Everything a dashboard sends shows up in the History below with the user's name.
  • You can turn all of this off with one switch in ⚙ Settings ("Allow dashboards to announce").

Tips

  • Save the announcements you repeat as quick messages (⚙ Settings) — one tap fills the composer.
  • Save your usual speakers + engine as defaults so the page is ready to send the moment it opens.
  • Use the master volume slider before a loud announcement, or per-tile sliders for a single room.
  • History has a ↻ button to re-send any past announcement.

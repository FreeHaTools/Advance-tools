🎛️ Which sensors does security use? ● Unsaved changes

Every sensor Home Assistant exposes is listed below. Turn Use off for anything security should ignore, then pick which modes watch it and whether it is instant or delayed. Nothing is saved until you press Save sensors.
Delayed = you get the entry countdown to type your code; use it for the door you walk in through. Instant = the alarm fires immediately; use it for windows and inside sensors.

⚡ What happens when the alarm goes off ● Unsaved changes

Pick anything you want to happen the moment the alarm triggers. Everything runs at once.
When triggered: nothing yet — pick some actions below.
📸 Take camera snapshots

Grabs a picture from every camera selected on the 📷 Cameras tab and attaches it to the alert notification.

🗣️ Announce out loud (text-to-speech)

Speaks a message on the speakers you choose. Great as a deterrent — an intruder hearing “the alarm has been triggered” usually leaves.

Keep it short — long sentences get cut off by some speakers.

⏱ Timings ● Unsaved changes

All three are in seconds.
How long you have to leave after arming Away before the system actually starts watching. Set it to how long it takes you to walk out and close the door.
How long you have to type your PIN after opening a sensor marked Delayed. If the countdown runs out, the alarm fires.
How long the sirens stay on after a trigger before switching themselves off. Many countries limit this to 180 seconds.

🔔 Where alerts are sent ● Unsaved changes

Tick every service that should receive the alarm message — you can pick as many as you like and they are all notified at the same time.
Want email or SMS? This list only shows notify services that already exist in Home Assistant. Advance Tools cannot create them for you. For email, add an SMTP notify integration in HA (Settings → Devices & services → Add integration → SMTP). For SMS, add an SMS gateway integration such as Twilio, or use a messaging app like Telegram. Once it exists, the new notify.… service appears in this list automatically — just reload this page.

How Security Center works

1
Create a PIN. One code arms, disarms and unlocks the dashboard keypad. It is 4 to 8 digits and you can change it any time from 🔢 PIN in the top bar.
2
Choose your sensors. On the 🎛️ Sensors tab every door, window, motion sensor and lock in your home is listed. Turn off the ones security should ignore (the shed door, a sensor you only use for automations), and tick which modes watch each one.
3
The three arming modes. 🏠 Home — you are inside, so normally only doors and windows are watched and you can move around freely. 🚗 Away — nobody is home, so everything is watched. 🌙 Night — you are asleep: perimeter plus the sensors downstairs, but not the ones on your way to the bathroom. What each mode watches is whatever you tick per sensor.
4
Exit and entry delay. After arming Away the exit delay counts down so you can leave without tripping your own alarm. When you come back and open a sensor marked Delayed, the entry delay counts down while you walk to the keypad and type your code. Sensors marked Instant give no grace period at all.
5
What triggers the alarm. A sensor that is in use, ticked for the current mode, and changes to open / detected while the system is armed. An instant sensor triggers immediately; a delayed one starts the entry countdown, and only triggers if the countdown reaches zero. The 🚨 Panic button triggers everything straight away, no delay.
6
What happens then. Everything you picked on the ⚡ Actions tab runs at once: sirens sound for the siren duration, lights and switches turn on, doors lock, scenes and scripts run, snapshots are taken and speakers announce your message. At the same time every notify service ticked on the 🔔 Alerts tab receives a message.
7
Everything is logged. The 📜 Log tab keeps the recent history — who armed, which sensor opened, when the alarm fired. Alerts are red.
⚠ Honest limitation. This is a convenience layer built on top of Home Assistant. It only works while Home Assistant and the Advance Tools add-on are running and your sensors are online — a power cut, a crashed add-on or a flat sensor battery means no alarm. There is no battery backup, no tamper detection and nobody is watching on the other end. It is not a certified or professionally monitored alarm system, and it should not be relied on as one.
